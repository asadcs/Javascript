1. For google chrome Javascript uses v8 engine which is used in node.js.
2. For mozilla Javascript uses Spider Monkey engine.
3. for Internet explorer Javascript uses Chakra.
4. Every engine has a parser that parse the javascript code.
5. when Error is genrated parser stops the execution.
6. If no error detected then code is converted to AST(Abstract Syntax Tree)
7. Now the AST is converted into Machine Code
